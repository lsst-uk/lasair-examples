## Notebooks for Lasair Kafka consumption

These notebooks show how to use Lasair Kafka.

You will need to clone this git repo and get an Lasair account and API key.
See [the documentation](https://lasair-lsst.readthedocs.io/en/develop/core_functions/python-notebooks.html)
for more details.

### consume.ipynb
Shows how to consume Kafka and print the results. You will need to change the `topic` to the one listed
in the webpage for your active filter.

### consume_plot.ipynb
Shows how to consume Kafka and plot the results. 
We assume the filter is saved with the option 'lightcurve_lite'.
You will need to change the `topic` to the one listed
in the webpage for your active filter.
